JMARS AR

Augmented and Virtual Reality Experiences for Planetary Research, Education, and Collaboration

Challenge

Configure JMARS, a free geospatial information system that provides access to planetary data, to be viewed in AR.

Project Team

Robert LiKamWa, Lauren Gold, Kathryn Powell, Hannah Bartolomea, Shaun Xiong, Justin Hertzberg, Alexander Gonzalez

My Role

Design UI

I was responsible for designing prototypes for this application using Figma and implenting the UI designs with front-end development in the Unity Game Engine.

Conduct User Studies

I worked with the graduate reserach assistant to conduct user studies via Zoom to ensure efficacy of the user interface.

Iterate Design

I continuously and adapted the user interface in response to user study results, co-worker recommendations, and client needs.

Design Plan

The intended audience was originally NASA scientists exclusively. It was difficult to understand how they intended to use the program and how to create navigation that would be most effective for them.

This design also underwent many iterations based on user observations and interviews. The top images show early developed versions and the bottom images show multiple screens of a later version, prototyped using Figma.

Final Design

The final prototype was designed in Adobe XD, as that was preferred by the client.

User observation showed some flaws in the previous protoype. This final design provides solutions to those issues, as described below.

Changes:

- Key decisions are displayed in a list, rather than a carousel which was determined to be confusing (3).

- Feedback is added to this current design to ensure that users are learning from each decision (4).

- Change an event page redesigned to give users more information about effects of each decision (6).

Project Outcome

NASA scientists, instrument members, and the public use JMARS AR to view planetary terrain in AR. It is available on the App Store and Google Play Store.

We envision JMARS AR Viewer to assist planetary scientists in communication and dissemination of their findings. We also see this a a good opportunity for public outreach, to increase support for space exploration.

This is my most recent project and is a culmination of my design and research experience during my undergraduate career.
